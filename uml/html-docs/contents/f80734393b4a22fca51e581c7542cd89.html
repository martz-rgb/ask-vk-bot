GetAnswersByDate

UMLOperation
AskBot :: Database :: Answers :: GetAnswersByDate

Description

Параметр from и\или параметр to могут быть равны null, -- тогда им выставляются значения по умолчанию. Значение по умолчанию параметра from = 2000-01-01T00:00:00Z. Значение по умолчанию параметра to = текущее время запроса.

Parameters

Direction Name Type Description
in from timestamp
in to timestamp
return Parameter1 Answers
